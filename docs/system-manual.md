## 第一章 概述

### 1.1 编写目的

本文档用于说明系统设计思路、功能边界、部署方式、操作步骤与维护要点，面向使用者、维护者和二次开发人员。

### 1.2 项目背景

针对新污染物环境数据中高维、非线性、多变量耦合问题，系统提供从数据清洗到模型解释的一体化分析能力，输出各影响因素相对重要性（RI），支持科研与管理决策参考。

### 1.3 系统定位

- 面向环境科研和管理场景的桌面分析工具
- 全图形化操作，降低机器学习使用门槛
- 以可解释性与可复现性为优先

### 1.4 适用场景

- 环境监测数据分析
- 污染物来源解析
- 风险评估辅助
- 科研论文图表与结果支撑
- 教学演示

## 第三章 系统架构设计

### 3.1 分层架构

- 表示层：各标签页 GUI
- 业务层：分析引擎、导出与报告
- 基础层：状态管理、事件总线、全局常量

### 3.2 核心文件

- app.py：主应用与页面装配
- launcher.py：启动闪屏
- state.py：应用状态
- event_bus.py：事件发布/订阅
- modules/analysis_engine.py：算法核心
- modules/tab_analysis.py：分析主流程

### 3.3 状态与线程

- 通过 AppState 管理全局状态
- 训练/验证/重计算在后台线程执行
- 使用主线程安全回调更新 UI

## 第四章 安装与部署

### 4.1 从源码运行

1. 安装 Python 3.10+
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 运行：

```bash
python run.py
```

### 4.2 打包部署

可使用 exe.txt 或 spec 文件打包：

```bash
pyinstaller --clean --noconfirm "新污染物分析系统.spec"
```

输出目录通常在 dist/ 下。

## 第五章 功能模块说明

### 5.1 数据模块

- 支持 xlsx/xls/csv/tsv
- 自动编码尝试与预览
- 同步刷新后续模块

### 5.2 特征与目标模块

- 勾选建模特征与目标变量
- 支持搜索、排序、类型设置与类别标签

### 5.3 预处理模块

- 缺失值处理、异常值处理
- 正态性检验与变换
- 标准化/归一化、过滤、PCA、交互特征

### 5.4 分析模块

- 多模型训练与评估
- Spearman 相关性
- 排列重要性与综合比较
- Excel 多工作表导出

### 5.5 CV 与图表模块

- K 折交叉验证与稳定性评估
- 热力图、重要性图、性能对比、SHAP、学习曲线

### 5.6 模拟模块

- 基于缓存模型进行 What-If 推演
- 通过滑块/下拉实时观察预测变化

## 第六章 操作流程指南

建议标准流程：

1. 导入数据
2. 查看统计概览
3. 选择特征与目标
4. 视需要进行预处理
5. 设置模型参数
6. 运行分析
7. 查看 CV 与图表
8. 进行情景模拟
9. 导出结果

## 第七章 数据格式规范

### 7.1 输入数据要求

- 第一行为列名
- 行为样本，列为变量
- 目标变量应可转为数值
- 分类特征可为文本标签

### 7.2 输出结果

- 模型结果表
- Spearman 结果与矩阵
- 模型性能汇总
- 因素重要性综合比较

### 7.3 配置文件

系统支持将分析配置保存为 JSON，便于复现。

## 第十章 常见问题与故障排除

1. 启动失败：优先检查依赖安装与日志
2. 分析耗时长：减少模型数或关闭自动调参
3. 图表异常：检查模型缓存和选择条件
4. 导出失败：检查目标路径权限与依赖
