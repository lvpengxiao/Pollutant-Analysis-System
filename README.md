## 📊 数据格式要求 (Data Requirements)

为了确保软件能够正常进行分析，请确保您导入的 Excel (`.xlsx`) 数据满足以下基本条件：
- **第一行必须是表头（列名）**，且列名最好不要包含特殊符号。
- 确保数据中没有大量的空值（NaN）或无法识别的非数字字符。
- 数据应为**宽表格式**：每一行代表一个独立的样本，每一列代表一种污染物特征或环境变量。
