\# 污染物分析系统



这是一个基于 CustomTkinter 和机器学习的多类新污染物室内分布影响因素高级分析程序。



\## 功能特点

\- 支持 RandomForest, XGBoost, LightGBM 等 6 种机器学习模型

\- 现代化的暗黑风格界面

\- 自动导出 Excel 综合结果报告



\## 如何使用

1\. 去 \[Releases](../../releases) 页面下载最新版的 `.zip` 压缩包

2\. 解压后，双击运行 `污染物分析系统.exe`

3\. 导入 Excel 数据即可开始分析
